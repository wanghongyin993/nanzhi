### 前言：

### 1.软件名称（单号扫码器）

> [!TIP]
>
> windons10以上系统

### 2.软件技术栈：

**技术栈**
**├── 核心框架: Tauri**
**├── 前端框架: Vue3 + TypeScript**
**├── 状态管理: Pinia**
**└── UI组件: Naive UI**

### 开发教程：

1. 打开软件目录进入danhao-app

2. 执行命令：

   ```node.js
   npm install
   ```

   3.安装插件成功后

   执行命令：

   ```node.js
    npm run tauri dev
   ```
